 Demo Description

 Key Features:
* **Auth Context:** Manages user sessions and persists login state using `localStorage`.
* **Theme Context:** A global toggle allowing users to switch between a clean **Light Mode** and a sleek **Dark Mode** (Tailwind CSS powered).
* **Api Context:** Handles asynchronous data fetching from an external API, managing loading states and global error handling.
* **Responsive Design:** Optimized for web viewing using **React Native Web** and **Tailwind CSS**.
* **Navigation:** Secure routing that prevents unauthenticated users from accessing the product list.
